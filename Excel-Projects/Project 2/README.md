# 📊 Data Science Salary Analysis in Microsoft Excel

An end-to-end Excel data analysis project that explores salary trends, in-demand skills, and job market insights for data professionals worldwide. The project leverages Pivot Tables, Pivot Charts, Slicers, Combo Charts, Scatter Plots, and Dashboarding techniques to transform raw job market data into interactive visualizations.

---

## 📌 Project Overview

This project analyzes global Data Science job market data to answer key business questions such as:

- Which data roles offer the highest salaries?
- Which technical skills are most in demand?
- How do salaries differ between the US and other countries?
- Is there a relationship between salary and the number of skills required?
- Which skills provide the highest earning potential?

The project concludes with an interactive Excel dashboard for exploring these insights.

---

## 🎯 Business Questions

- What are the highest-paying data-related jobs?
- Which skills are most requested by employers?
- Which technical skills command the highest salaries?
- How do salaries compare between the US and Non-US markets?
- Does having more required skills correlate with higher salaries?

# 📷 Project Analysis

## 1️⃣ Salary Comparison by Region

![Salary Analysis](Salary_Analysis.png)

### Insights

- Compares median salaries for major data roles across US and Non-US markets.
- Senior Data Scientist and Senior Data Engineer roles offer the highest salaries.
- Data Analyst and Business Analyst positions have comparatively lower salaries.
- Demonstrates regional salary differences across job roles.

---

## 2️⃣ Skills vs Salary Analysis

![Skills vs Salary](Salary_VS_Skills.png)

### Insights

- Examines the relationship between salary and the number of skills required.
- Higher-paying positions generally require broader technical skill sets.
- Senior Data Engineers require the highest average number of skills.
- A positive trend exists between salary and skill requirements.

## 4️⃣ Highest Paying Technical Skills

![Skill Salary](Skill_Salary_Analysis.png)
